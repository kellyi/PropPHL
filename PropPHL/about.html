About
ValPHL explores Philadelphia property data organized by block using the Philadelphia Property Data API and the Philly Hoods API.

Usage
Drop a pin on the map or type an address to search for property data from that block.

Tap the folder icon to browse through a list of blocks you've saved; tap the + icon from there search more blocks.
